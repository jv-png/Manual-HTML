TAGS

Oque são as Tags-

No HTML, tags são os comandos que dizem ao navegador como exibir o conteúdo de uma página. Elas sempre começam com <nome_da_tag> e, na maioria dos casos, terminam com </nome_da_tag>


Exemplos de Tags e para que servem:

  • <title> → Nome da aba do navegador.

  • <img> → Adiciona uma imagem.

  • <p> → Escreve um parágrafo.

  • <div> → Faz uma divisão no site.

  • <hr> → Cria uma linha horizontal.

  • <br> → Pula para a próxima linha.

  • <head> → Guarda informações do site (como o <title>).

  • <body> → Tudo que aparece na página.

  • <li> → Cria um item dentro de uma lista.

  • <ol> → Cria uma lista numerada (1, 2, 3...).

  • <ul> → Cria uma lista com pontos (•).

  • <button> → cria um botão clicável na página.

  • <input> → A tag é usada para criar campos interativos.

  • <h1> → Título principal (mais importante).

  • <h2> → Subtítulo de nível 2.

  • <h3> → Subtítulo de nível 3.

  • <h4> → Subtítulo de nível 4.

  • <h5> → Subtítulo de nível 5.

  • <h6> → Subtítulo de nível 6.

  • <table> → Cria uma tabela.

  • <thead> → Define o cabeçalho da tabela.

  • <tr> → Representa uma linha da tabela.

  • <th> → Define um cabeçalho de coluna dentro de <tr>

  • <tbody> → Agrupa o corpo da tabela.

  • <td> → Define uma célula de dado dentro de <tr>

  • <a> → É usada para criar links em HTML.

  • <form> → Cria um formulário em HTML para coletar dados do usuário, como textos, senhas e botões, e enviar essas informações para um servidor.

  • <form action> → Serve para definir o endereço (URL) para onde os dados do formulário serão enviados.

  • <form method> → Define como os dados do formulário serão enviados para o servidor.

  • Get e Post dentro da tag <form> → Get-Envia os dados pela URL (visível) Ou seja quando não tem dados sensíveis. e Post- Envia os dados escondidos no corpo da requisição, ou seja quando tem dados sensíveis.

  • <label> → Serve para criar rótulos (legendas) que identificam campos de um formulário, como caixas de texto, botões ou checkboxes.

  • <select> → É usada para criar caixas de seleção (dropdown) em formulários HTML, onde o usuário pode escolher uma ou mais opções.

  • <option> → é usada dentro da tag <select> para criar as opções da lista de seleção (dropdown).

    • Tag <input>

    A tag <input> possui atributos que podem ser utilizados na contrução de um formulário.


    Os atributos type, name e value na tag <input> servem para definir como o campo de entrada vai funcionar no formulário, exemplo:


    type:Define o tipo de entrada

    name:Dá um nome único para identificar o campo na hora de enviar os dados

    value:Define o valor que já vai aparecer preenchido ou o valor enviado

    CSS-

    Oque é CSS? E qual a sua função?-

    CSS é a linguagem usada para estilizar páginas da web.
    Enquanto o HTML define o conteúdo e a estrutura (títulos, parágrafos, imagens...), o CSS define a aparência — como cores, tamanhos, espaçamentos, fontes e o layout.

    Oque são seletores CSS?

    Seletores CSS são usados para "apontar" quais elementos HTML você quer estilizar.

    Um exemplo de seletor CSS é a classe

    Classe no CSS é uma forma de dar um nome a um grupo de estilos e aplicar esse nome a qualquer elemento HTML que você quiser.

    Como funciona-

    HTML:

    <p class="destaque">Texto importante!/p>

    CSS:

    .destaque {
    color: red;
    font-weight: bold;
    }

    Principais comandos em CSS:

  • background-color → Define a cor de fundo do elemento.

  • font-size → Define o tamanho da fonte.

  • font-family → Define o tipo de fonte usada no texto.

  • border → Cria uma borda ao redor do elemento.

  • margin: 0 auto; → Centraliza um elemento horizontalmente dentro do pai.

  • height → Define a altura do elemento.

  • padding → Define o espaço interno entre o conteúdo e a borda.

  • display: flex; → Ativa o Flexbox, um sistema para organizar os elementos dentro de um contêiner.
